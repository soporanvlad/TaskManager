# TaskManager

A desktop task management application developed in Python using CustomTkinter and SQLite. The project follows a layered architecture and provides task organization, filtering and status tracking functionalities.

## Features

- User authentication
- Create, edit and delete tasks
- Mark tasks as completed or pending
- Filter tasks by category, priority and status
- Task categorization
- SQLite database persistence
- Graphical user interface built with CustomTkinter

## Technologies Used

- Python
- CustomTkinter
- SQLite
- Object-Oriented Programming (OOP)

## Architecture

The application follows a layered architecture consisting of:

- UI Layer – graphical user interface and user interactions
- Service Layer – business logic
- Repository Layer – data access and persistence
- Entity Layer – domain models
